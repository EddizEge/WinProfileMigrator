# WinProfileMigrator - Windows Profil Taşıma & Klonlama Aracı

## 🚀 Özellikler

- **Profil Dışa Aktarma**: Seçili profil bileşenlerini ZIP paketi olarak dışa aktarın
- **Profil İçe Aktarma**: Daha önce dışa aktarılan profilleri geri yükleyin
- **Ağ Transferi**: İki bilgisayar arasında doğrudan ağ üzerinden profil aktarın
- **Profil Klonlama**: Mevcut profili kopyalayarak yeni yerel profil oluşturun

## 📦 Taşınabilir Bileşenler

- Masaüstü, Belgeler, İndirilenler, Resimler, Müzik, Videolar
- Uygulama ayarları (AppData)
- Registry ayarları (HKCU)
- Wi-Fi profilleri
- Yazıcı ayarları
- Ortam değişkenleri
- Duvar kağıdı ve tema ayarları
- Görev çubuğu ve başlat menüsü pinleri

## 🛠️ Kurulum

```bash
pip install -r requirements.txt
```

## ▶️ Çalıştırma

```bash
python main.py
```

> **Not**: Bazı özellikler yönetici (Administrator) yetkisi gerektirir.

## 📋 Gereksinimler

- Windows 10/11
- Python 3.9+
- Yönetici yetkileri (tam işlevsellik için)
